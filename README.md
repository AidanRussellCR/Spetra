# Spetra

<img width="330" height="120" alt="spetra_logo" src="https://github.com/user-attachments/assets/323cc1da-a245-4e9c-bb40-45d0b7571508" />

A lightweight game engine for performance and convenience.

Spetra, meaning "Spear" in the Thurlsk language, is meant to represent the lightweight, convenient, and efficient engine that Spetra aims to be.
While not intending to compete per say with other engines out there, Spetra is being built with an aim to at least help one person in some way.

## Current Progress
- Window + Rendering system
- Input handling
- Delta-time system
- Scene system + scene switching
- Texture loading
- Tile-based map rendering
- Player movement (delta-time based)
- Tile-based collision system
- Camera (still very basic)

## Roadmap
- Map System
- Map loading/saving
- Map Editor
- Camera follow + smoothing
- Tile-based interaction triggers
- NPC system
- Event scripting system
- Battle system (turn-based)
- Animated tiles
- Partial tile collision (slopes, masks)
- Parallax layers
- Lighting system
